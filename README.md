# educational-pr# SDR Signal Processing

Проект по учебной практике, реализующий обработку IQ-сигнала.

## Возможности

- чтение IQ-сигнала из бинарного файла;
- AM-демодуляция;
- FM-демодуляция;
- нерекурсивный SMA-фильтр;
- рекурсивный SMA-фильтр;
- сохранение результата обработки;
- модульные тесты с использованием Google Test.

## Структура проекта


```
matlab файл
ConsoleApplication1/
    ConsoleApplication1/
        Основной исходный код проекта
        matlab файл для тестирования
    
    Tests/
        Модульные тесты
```

## Используемые технологии

- C++
- CMake
- Google Test
- Visual Studio 2022
- matlab
