# educational-pr# SDR Signal Processing

Проект по учебной практике, реализующий обработку IQ-сигнала.

## Возможности

- чтение IQ-сигнала из бинарного файла;
- AM-демодуляция;
- FM-демодуляция;
- нерекурсивный SMA-фильтр;
- рекурсивный SMA-фильтр;
- сохранение результата обработки;
- модульные тесты с использованием Google Test.

## Структура проекта


```
matlab файл
ConsoleApplication1/
    Основной исходный код проекта
    matlab файл для тестирования

Tests/
    Модульные тесты
```

## Используемые технологии

- C++
- CMake
- Google Test
- Visual Studio 2022

## Ограничения

По техническим причинам исходный файл FM сигнала (`.bin`) отсутствует в репозитории. Он весит более 25мб.

Для запуска программы необходимо самостоятельно поместить бинарный файл с IQ-данными в рабочую директорию проекта.

В текущей реализации отсутствие файла не обрабатывается отдельным сообщением об ошибке, поэтому при его отсутствии программа не сможет выполнить загрузку сигнала.
